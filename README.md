## Requirements
- Node >= 20
- pnpm
- postgresql

## Development
- Run `./scripts/reset-db` to setup database and seed
- Run `./scripts/dev` to start development server

## Testing
- Run `./scripts/test` to start test suites

## Tools
- ESlint
- Prettier
